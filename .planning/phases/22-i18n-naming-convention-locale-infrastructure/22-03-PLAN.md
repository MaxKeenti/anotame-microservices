<action>
Restructure `hooks.server.ts` to wrap the existing proxy logic inside `paraglideMiddleware`. The locale detection chain is:
1. Read `locale` cookie → if valid ("es" or "en"), use it
2. No cookie → `Accept-Language` header (for anonymous pages)
3. Final fallback → "es" (sourceLanguageTag)

Replace the current `export const handle` with:

```typescript
import type { Handle } from '@sveltejs/kit';
import { env } from '$env/dynamic/public';
import { paraglideMiddleware } from '$lib/paraglide/server';

function resolveBackendUrl(apiPath: string): string | null {
  // ... existing function unchanged ...
}

export const handle: Handle = async ({ event, resolve }) => {
  const { request } = event;
  const url = new URL(request.url);

  // API proxy requests do NOT need locale resolution — short-circuit
  if (url.pathname.startsWith('/api/')) {
    // ... existing proxy logic unchanged (lines 54-150 of current file) ...
  }

  // Non-API requests: apply Paraglide locale middleware
  return paraglideMiddleware(event.request, ({ request: localizedRequest, locale }) => {
    event.request = localizedRequest;

    return resolve(event, {
      transformPageChunk: ({ html }) => html.replace('%lang%', locale),
    });
  });
};
```

**Critical implementation detail:** The API proxy block must remain as a standalone early-return path BEFORE the `paraglideMiddleware` call. API requests don't render HTML and don't need locale resolution. This keeps the proxy fast and avoids `AsyncLocalStorage` overhead for API calls.

```typescript
  // setLocale is called by paraglideMiddleware internally,
  // but if cookie strategy isn't auto-detected, set it here
}
```

The exact integration depends on what `$lib/paraglide/server.js` exports after generation — read that file first before implementing.

<title>Add locale to User interface and cookie management in AuthService</title>
<read_first>
- anotame-web/src/lib/services/auth.svelte.ts
- anotame-web/src/lib/services/api.svelte.ts

<action>
1. Update the `User` interface in `auth.svelte.ts` to add locale:
   ```typescript
   export interface User {
       id: string;
       username: string;
       email: string;
       role: string;
       locale: string;
   }
   ```

2. Add a private helper to set the locale cookie:
   ```typescript
   private setLocaleCookie(locale: string) {
       if (typeof document !== 'undefined') {
           document.cookie = `locale=${locale};path=/;max-age=${60 * 60 * 24 * 365};SameSite=Lax`;
       }
   }
   ```

3. In the `login()` method, after `this.userState.current = response;`, add:
   ```typescript
   this.setLocaleCookie(response.locale || 'es');
   ```

4. Add a `changeLocale()` method to AuthService:
   ```typescript
   async changeLocale(newLocale: string): Promise<void> {
       if (!this.user) return;

       await apiService.request(`${API_IDENTITY}/users/${this.user.id}/locale`, {
           method: 'PATCH',
           body: JSON.stringify({ locale: newLocale })
       });

       this.userState.current = { ...this.user, locale: newLocale };
       this.setLocaleCookie(newLocale);
   }
   ```

5. Add a `clearLocaleCookie()` call in `logout()`, before the redirect:
   ```typescript
   if (typeof document !== 'undefined') {
       document.cookie = 'locale=;path=/;max-age=0';
   }
   ```

6. Import `API_IDENTITY` if not already imported (it's already imported via `apiService`'s module — check the imports).
</action>

- `logout()` clears the locale cookie

2. Add locale change handler function:
   ```typescript
   let changingLocale = $state(false);

   async function handleLocaleChange(newLocale: string) {
     if (newLocale === getLocale()) return;
     changingLocale = true;
     try {
       await authService.changeLocale(newLocale);
       // Paraglide soft swap — reload translations without full page reload
       const { setLocale } = await import('$lib/paraglide/runtime');
       setLocale(newLocale);
       await invalidateAll();
       toast.success(newLocale === 'en' ? 'Language changed to English' : 'Idioma cambiado a Español');
     } catch (e: any) {
       toast.error(e.message || 'Error al cambiar idioma');
     } finally {
       changingLocale = false;
     }
   }
   ```
