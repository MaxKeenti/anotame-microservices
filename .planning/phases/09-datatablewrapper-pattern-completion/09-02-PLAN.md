# Plan 09-02: Migrate Services and Price Lists Pages to DataTableWrapper

<objective>
Replace raw `Table.*` markup in the Services and Price Lists pages with DataTableWrapper. These pages are slightly more complex — Services has an external filter panel (garment dropdown) that must be preserved, and Price Lists has a Card wrapper and status badges.
</objective>

<must_haves>
- Services page renders via DataTableWrapper with sorting on Name column
- Services page retains its external filter panel (search + garment dropdown) passing filtered data to DataTableWrapper
- Price Lists page renders via DataTableWrapper inside the existing Card wrapper
- Both pages retain their action buttons (Edit/Delete for Services, Clone/View/Delete for Price Lists)
- No raw Table.* markup remains
- `bun run build` exits 0
</must_haves>

---

## Tasks

<task id="09-02-01">
<title>Migrate Services page to DataTableWrapper</title>

<read_first>
- anotame-web/src/routes/(app)/dashboard/catalog/services/+page.svelte
- anotame-web/src/lib/components/ui/DataTableWrapper.svelte
- anotame-web/src/routes/(app)/dashboard/orders/+page.svelte (reference — external filters + DataTableWrapper)
</read_first>

<action>
1. Add imports at top of script block:
   ```
   import DataTableWrapper from '$lib/components/ui/DataTableWrapper.svelte';
   import type { ColumnDef } from '@tanstack/table-core';
   ```

2. Remove import of `* as Table` from `'$lib/components/ui/table'`.

3. Define columns using `$derived` (since `isAdmin` is reactive):
   ```typescript
   let columns = $derived<ColumnDef<any>[]>([
     { accessorKey: 'name', header: 'Nombre', enableSorting: true },
     { id: 'garment', accessorFn: (row: any) => getGarmentName(row.garmentTypeId), header: 'Prenda', enableSorting: true },
     { accessorKey: 'defaultDurationMin', header: 'Duración (min)', enableSorting: true },
     { id: 'price', accessorFn: (row: any) => `$${row.basePrice.toFixed(2)}`, header: 'Precio', enableSorting: true },
     ...(isAdmin ? [{ id: 'actions', header: 'Acciones', enableSorting: false } as ColumnDef<any>] : []),
   ]);
   ```

4. KEEP the external filter panel (search input + garment AdaptiveSelect) — it provides domain-specific filtering that goes beyond DataTableWrapper's global search.

5. KEEP the `filteredServices` derived — pass it as data to DataTableWrapper.

6. Replace the raw table block (lines 144-204) with:
   ```svelte
   <div class="bg-card border border-border rounded-xl overflow-hidden shadow-sm p-4">
     <DataTableWrapper
       {columns}
       data={filteredServices}
       {loading}
       emptyMessage="No se encontraron servicios."
       filterPlaceholder="Filtrar servicios..."
     >
       {#snippet actionCell(row)}
         <div class="flex justify-end gap-2">
           <Button
             variant="outline"
             size="sm"
             class="h-10 px-4 touch-manipulation font-medium"
             onclick={() => handleEditClick(row.original)}
           >
             <Edit class="w-4 h-4 mr-2" />
             Editar
           </Button>
           <Button
             variant="outline"
             size="sm"
             class="h-10 px-4 text-destructive hover:text-destructive/90 touch-manipulation font-medium"
             onclick={() => handleDeleteClick(row.original)}
           >
             <Trash2 class="w-4 h-4 mr-2" />
             Eliminar
           </Button>
         </div>
       {/snippet}
     </DataTableWrapper>
   </div>
   ```

7. Keep `ServiceDialog` component at the bottom unchanged.
8. Keep `getGarmentName` helper function — it is used in the column accessorFn.
</action>

<task id="09-02-02">
<title>Migrate Price Lists page to DataTableWrapper</title>

<read_first>
- anotame-web/src/routes/(app)/dashboard/catalog/pricelists/+page.svelte
- anotame-web/src/lib/components/ui/DataTableWrapper.svelte
- anotame-web/src/routes/(app)/dashboard/customers/+page.svelte (reference implementation)
</read_first>

<action>
1. Add imports at top of script block:
   ```
   import DataTableWrapper from '$lib/components/ui/DataTableWrapper.svelte';
   import type { ColumnDef } from '@tanstack/table-core';
   ```

2. Remove import of `* as Table` from `'$lib/components/ui/table'`.

3. Define columns after imports:
   ```typescript
   const columns: ColumnDef<any>[] = [
     { accessorKey: 'name', header: 'Nombre', enableSorting: true },
     { accessorKey: 'priority', header: 'Prioridad', enableSorting: true },
     { id: 'validFrom', accessorFn: (row) => new Date(row.validFrom).toLocaleDateString('es-ES'), header: 'Válido Desde', enableSorting: true },
     { id: 'validTo', accessorFn: (row) => row.validTo ? new Date(row.validTo).toLocaleDateString('es-ES') : 'Permanente', header: 'Válido Hasta', enableSorting: true },
     { id: 'status', accessorFn: (row) => row.active ? 'Activa' : 'Inactiva', header: 'Estado', enableSorting: true },
     { id: 'actions', header: 'Acciones', enableSorting: false },
   ];
   ```
   NOTE: Status badges become plain text ("Activa"/"Inactiva"). This is acceptable per research decision — the DataTableWrapper does not support custom cell renderers for non-action columns.

4. Replace the entire Card.Content inner block — keep the Card.Root wrapper but replace the raw table inside it. The template should become:
   ```svelte
   <Card.Root>
     <Card.Header>
       <Card.Title>Estrategias Activas e Históricas</Card.Title>
       <Card.Description>Las listas con mayor prioridad sobrescriben el precio base del catálogo.</Card.Description>
     </Card.Header>
     <Card.Content>
       <DataTableWrapper
         {columns}
         data={lists}
         loading={isLoading}
         emptyMessage="No hay listas configuradas."
         filterPlaceholder="Buscar listas..."
       >
         {#snippet actionCell(row)}
           <div class="flex justify-end gap-2">
             <Button
               variant="outline"
               size="sm"
               class="h-10 border-primary/20 hover:bg-primary/5 text-primary"
               onclick={() => handleClone(row.original.id)}
               title="Clonar Lista"
             >
               <Copy class="w-4 h-4 mr-2" />
               Clonar
             </Button>
             <Button
               variant="outline"
               size="sm"
               class="h-10"
               href={`/dashboard/catalog/pricelists/${row.original.id}`}
             >
               <Eye class="w-4 h-4 mr-2" />
               Ver
             </Button>
             <Button
               variant="outline"
               size="sm"
               class="h-10 text-destructive hover:bg-destructive/10 border-destructive/20"
               onclick={() => handleDelete(row.original.id, row.original.name)}
             >
               <Trash2 class="w-4 h-4 mr-2" />
               Eliminar
             </Button>
           </div>
         {/snippet}
       </DataTableWrapper>
     </Card.Content>
   </Card.Root>
   ```

5. Remove the manual `{#if isLoading}` / `{#each}` / empty state blocks — DataTableWrapper handles all of these.

6. Keep `useAuthGuard`, `guard.checking` / `guard.allowed` template conditionals unchanged — this is a page-level concern.
</action>
