The file must contain:

1. `PAGE_SIZE_OPTIONS = [5, 10, 20, 50] as const` — exported const tuple (D-03)

2. A private `_pageSize = new PersistedState<number>('table_page_size', 20)` — localStorage
   key is exactly `'table_page_size'`, default is `20` (D-01, D-02)

3. An exported `tablePreferences` singleton object with:
   - `get pageSize(): number` — reads `_pageSize.current`, validates it against
     `PAGE_SIZE_OPTIONS`, returns it if valid, otherwise returns `20` as fallback
     (guards against stale/corrupt localStorage — D-03)
   - `setPageSize(n: number): void` — sets `_pageSize.current = n` only if `n` is in
     `PAGE_SIZE_OPTIONS`; silently ignores invalid values

Do NOT import `authService` — no user scoping.
Do NOT add any other methods or state beyond what is listed.

Resulting file (exact shape):
```typescript
import { PersistedState } from 'runed';

export const PAGE_SIZE_OPTIONS = [5, 10, 20, 50] as const;
type PageSizeOption = (typeof PAGE_SIZE_OPTIONS)[number];

const _pageSize = new PersistedState<number>('table_page_size', 20);

export const tablePreferences = {
  get pageSize(): number {
    const stored = _pageSize.current;
    return (PAGE_SIZE_OPTIONS as readonly number[]).includes(stored) ? stored : 20;
  },
  setPageSize(n: number): void {
    if ((PAGE_SIZE_OPTIONS as readonly number[]).includes(n)) {
      _pageSize.current = n as PageSizeOption;
    }
  },
};
```
