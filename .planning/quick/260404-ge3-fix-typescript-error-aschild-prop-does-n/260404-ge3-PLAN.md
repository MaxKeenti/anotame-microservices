<context>
The formsnap library's Label component does NOT support the `asChild` prop pattern. Unlike bits-ui components which use `asChild` to opt out of rendering, formsnap's Label uses the Svelte snippet pattern (via `{#snippet child}`) for customization.

The form-label.svelte component is already using formsnap's snippet pattern correctly to wrap the base Label component. The `asChild` prop is unnecessary and incompatible with formsnap's API.

**Current code (broken):**
```svelte
<FormPrimitive.Label asChild {...restProps} bind:ref>
	{#snippet child({ props })}
		<Label {...props} data-slot="form-label" class={cn(...)} >
			{@render children?.()}
		</Label>
	{/snippet}
</FormPrimitive.Label>
```

**Fix:** Remove the `asChild` prop since formsnap already supports snippet-based rendering.
</context>
