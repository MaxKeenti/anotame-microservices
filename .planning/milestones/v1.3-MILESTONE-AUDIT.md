## Cross-Phase Integration

### Integration Point 1: DataTableWrapper (Phase 15-03 × Phase 17-01)

Both phases modified `DataTableWrapper.svelte`. Phase 15-03 added bulk mode props and the `__select__` checkbox column. Phase 17-01 added the `tablePreferences` import and changed the `initialPageSize` intercept line.

**Verified live:**
- Line 3: `import { tablePreferences } from '$lib/stores/table-preferences.svelte';` ✓
- Line 31–33: `bulkActions?: boolean`, `bulkMode?: boolean`, `onSelectionChange?` props ✓
- Line 47–48: `bulkMode = $bindable(false)`, `onSelectionChange` in `$props()` ✓
- Line 52: `let initialPageSize = untrack(() => tablePreferences.pageSize);` ✓

Both change sets coexist without conflict. The bulk mode and row count features are orthogonal — bulk mode controls row selection; row count controls pagination page size.

### Integration Point 2: OrderResponse DTO (Phase 15 × Phase 16)

Phase 15 added `pickupCode` and `deliveredAt` to `OrderResponse`; Phase 16 added `priceListId` and `priceListName`. All four fields must coexist in `dtos.ts`.

**Verified live in `anotame-web/src/lib/types/dtos.ts`:**

All four fields present. ✓

### Integration Point 3: Edit Wizard reads Phase 16 data (Phase 15-02 × Phase 16)

Phase 16 VERIFICATION confirms: "edit wizard shows order's price list name as read-only (non-interactive badge) — VERIFIED". ✓
