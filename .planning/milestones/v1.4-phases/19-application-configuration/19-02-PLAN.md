| T-19-06 | Information Disclosure | operations-service application.properties (jdbc.url) | mitigate | Same as T-19-05 for operations-service |

| T-19-08 | Denial of Service | quarkus.http.port (hardcoded) | mitigate | Change to `${PORT:8083}` / `${PORT:8084}` — Railway's PORT assignment always honored when set |
