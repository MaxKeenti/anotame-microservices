# Strings-Only Localization

Anotame localizes UI strings through Paraglide while keeping dates, numbers, and currency in the Mexico business format. The shop operates in Mexico with MXN and DD/MM/YYYY expectations, so English UI text does not imply locale-aware financial or date formatting.

**Consequences:** Spanish is the authoritative business language, English translations can be rough, and locale work should focus on message extraction unless a future milestone explicitly changes formatting policy.
